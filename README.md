# CodTech Internship Task-4: Sentiment Analysis

## Objective:
Perform sentiment analysis on textual data (tweets/reviews) using NLP techniques.

Model Accuracy ~80% with insightful visualizations

Python, Pandas, scikit-learn, NLTK, WordCloud, Matplotlib, Seaborn
